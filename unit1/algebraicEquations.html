Unit 1: Algebraic Equations


The first topic that you will learn is algebraic equations. In these equations, you are trying to solve for the variable x. A variable is just a letter that is substituted for a number. The variable doesn't have to be x, but it is most commonly used. We will start with simple algebraic equations and then we will move onto multi step equations

Here is a simple equation:


2x + 10 = 14


To solve this equation, the first step would be to subtract 10 from both sides of the equation, so we can isolate the variable with the coeficciant(If it was 2x-10, you would just add ten to both sides). Whatever you do to one side of the equation, must be done to the other side of the equation.


2x = 4


Now that we have 2x = 4, we need to isolate x. To do this we have to divide the whole equation by 2.

2x/2= 4/2

x=2


Therefore, the answer to this equation is 2. This is how you do equations. You have to solve for whatever variable is given. We will continue with more advanced forms of equations that involve more steps.

Here are some practice problems that you can do on a seperate sheet of paper:

  1. 5x + 31 = 106

  2. 3x + 6 = 36

  3. 10x - 4 = 96

  4. 4x - 7 = 21

  5. x + 4 = 22





Here are the answers:

  1. x = 15

  2. x = 10

  3. x = 10

  4. x = 7

  5. x = 18


Multi Step Equations

The next part of this unit that you will learn is multi step equations. Multi step equations are just like normal equations, but they have multiple steps. For example, 4(x-5)+3x = 4+3x is a multi step equations because it includes order of operations

To solve this equation, we have to break it up into parts. The first step would be to simplify all the parenthesis.


4x-20+3x = 4+3x


Then we combine like terms.


7x-20 = 4+3x


Then we would do what we normally would in a equation.


4x=16


The final step would be to divide by 4.


x=4


The solution to the equation is x=4. You can plug 4 back into the equation to make sure you got the answer right.
