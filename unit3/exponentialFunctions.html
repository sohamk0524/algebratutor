Unit 3: Exponential Functions

The next topic we will discuss is exponential functions. As you can tell by the title, exponential functions increase exponentially. An exponential function is made when a value in a data table increases exponentially and not in a constant rate of change. Data tables are used for exponential functions. Here is an example of a data table for an exponential function

src="https://i.ytimg.com/vi/d88fCE6W2sM/hqdefault.jpg">

As you can see in the image, the exponential function is increasing by a factor of 4. The table doesn't show a constant rate of change. The equation for an exponential function is y=a*b^x

Take note that if b > 1 then the graph will open right and if b < 1 it will open to the left. In the equation, "a" is the y-intercept.


Here is an example of graph:

In this graph, you can see that the line passes through 1 in the y axis. That means a is 1. If a is one, you don't have to put anything. Also,since the graph is opening to the right, that means b > 1.

To figure out b, you have to take two points on the line and see what is the ratio or factor that the values increase by. In this graph, two points are (1,2) and (2,4).
